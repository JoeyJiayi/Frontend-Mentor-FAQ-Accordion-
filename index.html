FAQs

What is Frontend Mentor, and how will it help me?

Frontend Mentor offers realistic coding challenges to help developers improve their frontend coding skills with projects in HTML, CSS, and JavaScript. It's suitable for all levels and ideal for portfolio building.

Is Frontend Mentor free?

The majority of the challenges are free. There are premium challenges which require a Pro subscription to access. It will say on each challenge whether they are free or premium.

Can I use Frontend Mentor projects in my portfolio?

Definitely! Please feel free to use whatever you build in your portfolio.

How can I get help if I'm stuck on a challenge?

The best (and quickest) way to get help on a challenge is in our Discord server. There are thousands of other developers in there, so it's a great place to ask questions. We even have a dedicated "help" channel!
